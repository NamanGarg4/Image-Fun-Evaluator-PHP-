# Image-Fun-Evaluator-PHP-
A simple PHP website where users can upload an image and get ratings (1–10) for different categories like **Scary, Funny, Cute, Weird**, and more. Ratings are randomized and displayed with stars.

Features
1. Upload images (JPG, PNG, GIF).
2. Select one or multiple categories to evaluate.
3. Get random ratings for each selected category.
4. Stylish display with stars.
5. Built with PHP, HTML, CSS, and JS (no external APIs).

How to Run
1. Install PHP or use a local server (XAMPP/WAMP/MAMP).
2. Place the project folder in your server’s root directory.
3. Open in browser: `http://localhost/image_rating/index.php`
4. Upload an image, select categories, and click Evaluate

Notes
1. Only JPG, PNG, GIF images are allowed.
2. Ratings are random for each category.
3. Images are saved in the `uploads/` folder.
